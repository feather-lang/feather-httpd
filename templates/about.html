Feather is a lightweight, embeddable scripting language with Tcl-like syntax. It's designed to be easy to integrate into host applications—like this HTTP server.

Why Feather?

  • Simple syntax — commands and arguments, no complex grammar
  • Easy embedding — expose Go functions directly to scripts
  • Minimal footprint — small interpreter, fast startup
  • Extensible — add custom commands for your domain

Why Embed a Scripting Language?

Embedding Feather gives you the best of both worlds:

  • Runtime control — modify server behavior without restarting
  • Live introspection — inspect running state like active SSE connections
  • Simple DSL — configure routes and handlers in a readable script
  • Performance where it matters — Go handles HTTP, templates, and concurrency
  • User-scriptable — Feather provides a thin control layer anyone can customize

About This Demo

The Go code provides the infrastructure:

  • HTTP server and routing
  • Template rendering
  • SSE (Server-Sent Events) support
  • Request/response handling primitives

Feather scripts wire it all together—defining routes, setting headers, rendering templates, and managing real-time connections.
